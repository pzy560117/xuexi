# Midscene v0.x 核心版本 Release Notes

> v0.x 系列奠定了 Midscene 的技术基础，从 Web 自动化扩展到多平台

---

## v0.30.0 — 缓存管理升级（2025-09-30）
- **灵活缓存策略**: 支持只读模式和灵活的缓存选项
- **默认读写策略**: 添加默认读写策略和 xpath 缓存默认值
- **runAdbCommand**: Android Agent 新增 ADB 命令方法
- **iOS 设备像素比**: 实现正确的设备像素比检测

## v0.29.0 — iOS 平台支持（2025-09-24）
- **iOS 自动化**: 通过 WebDriver 支持 iOS 设备
- **Qwen3-VL 适配**: 适配通义千问 3 视觉语言模型
- **UI-TARS 改进**: 改进 aiAction + UI-TARS 实现
- **aiAssert 更新**: 更新断言提示词

## v0.28.0 — 自定义接口集成（2025-09-08）
- **自定义接口**: 支持集成任何界面（预览功能）
- **双击动作**: 添加双击操作支持
- **Android 规划缓存**: 支持 Android 平台的规划缓存
- **PlaygroundSDK**: 实现执行管理 SDK
- **Hook 系统**: 添加 Action Space 钩子
- **长按和滑动**: Web 端长按和滑动手势

## v0.27.0 — 核心模块重构（2025-08-21）
- **Action Space**: 使用 Zod schema 定义动作参数
- **模型配置隔离**: 创建 Agent 时支持覆盖模型配置
- **图片提示**: aiWaitFor 支持图片提示
- **多模型报告**: 报告中显示多模型信息

## v0.26.0 — Rslib 构建升级（2025-08-11）
- **Rslib 构建**: 工具链完全集成 Rslib
- **上下文复用**: Web 集成支持复用上下文参数
- **懒解析报告**: 延迟解析报告内容提升性能

## v0.25.0 — 图片提示（2025-08-05）
- **图片提示输入**: 支持使用图片作为 AI 提示输入
- **规划缓存先行**: 缓存文件中先保存规划
- **photon + sharp**: 使用 photon 和 sharp 进行图片缩放
- **Chrome 录制器**: 支持导出所有事件到文档
- **坐标获取 XPath**: 通过坐标获取 xpath

## v0.24.0 — MCP for Android（2025-07-28）
- **Android MCP**: 支持 Android 工具的 MCP 服务
- **通用 Playground**: 实现通用 Playground

## v0.23.0 — 全新报告样式（2025-07-21）
- **新报告 UI**: 全新报告样式设计
- **YAML 配置模式**: 支持多 YAML 文件执行
- **Chrome 录制器**: 发布 Chrome 扩展录制功能

## v0.22.0 — Chrome 录制功能（2025-07-16）
- **录制器**: Chrome 扩展新增录制功能
- **IndexedDB**: Playground 和扩展使用 IndexedDB
- **最大重试限制**: 允许配置最大回收限制

## v0.21.0 — Chrome 扩展 UI 升级（2025-07-04）
- **全新 Chrome 扩展 UI**: 重新设计扩展界面
- **onDumpUpdate 回调**: 任务执行后回调
- **超时覆盖**: fixture 支持覆盖超时

## v0.20.0 — XPath 定位（2025-06-19）
- **aiAsk**: Agent 支持 aiAsk 方法
- **XPath 选项**: 支持用户期望 xpath 选项定位

## v0.18.0 — 增强报告（2025）
- **自定义报告节点**: 报告中添加自定义节点
- **视频导出**: 支持将报告下载为视频

## v0.17.0 — DOM 可见性（2025）
- **AI 看 DOM**: 让 AI 看到页面的 DOM
- **右键操作**: 新增右键点击能力

## v0.16.0 — MCP 支持（2025）
- **Midscene MCP**: 首次支持 Model Context Protocol
- **结构化 API**: Agent 支持结构化 API

## v0.15.0 — Android 自动化（2025）

## v0.14.0 — Instant Actions（2025）
