# 相机直连手机 + AI 精修 完整工作流

## 概述

本文档总结了从相机拍照到手机 AI 精修的完整流程，涵盖设备连接、照片传输、AI 修图三个核心环节。

---

## 一、相机到手机的实时传输方案

### 各品牌官方 App

| 相机品牌 | 官方 App | 传输方式 | 自动传输 |
|---------|---------|---------|---------|
| 佳能 | Canon Camera Connect | Wi-Fi / 蓝牙 | ✅ 拍完自动传 |
| 尼康 | Nikon SnapBridge | 蓝牙低功耗 | ✅ 后台自动传 |
| 索尼 | Sony Imaging Edge Mobile | Wi-Fi | ✅ 拍完自动传 |
| 富士 | FUJIFILM Camera Remote | 蓝牙 / Wi-Fi | ✅ 拍完自动传 |
| 松下 | Panasonic Image App | Wi-Fi / 蓝牙 | ✅ 拍完自动传 |
| 奥之心 | OM System | Wi-Fi | ❌ 手动选择传 |

### 设置步骤（以佳能为例）

1. 相机菜单 → **Wi-Fi/蓝牙设置** → 开启蓝牙
2. 手机安装 **Canon Camera Connect**，完成配对连接
3. 相机设置 → **拍摄后自动传输** → 开启
4. 每拍一张照片，JPEG 会自动推送到手机相册

### 注意事项

- **蓝牙自动传输**：通常只传 JPEG 小图（约 2~3MB），适合快速预览和社交分享
- **Wi-Fi 传输**：可传原图甚至 RAW，但需手动选择，速度更快
- **建议**：日常出片用蓝牙自动传 JPEG，商业用途用 Wi-Fi 传 RAW

## 三、醒图 AI 精修详细操作指南

### 核心功能入口

| 功能 | 操作路径 | 适用场景 |
|------|---------|---------|
| AI 一键精修 | 首页 → 选照片 → 底部「智能」→「一键精修」 | 快速出片，自动调色调光 |
| AI 消除 | 编辑 → 底部「修复」→ 涂抹要消除的区域 | 去路人、去杂物、去水印 |
| AI 换天 | 编辑 →「创意」→「天空」→ 选模板 | 阴天变晴天、加晚霞 |
| AI 人像精修 | 编辑 →「人像」→ 自动识别人脸 | 磨皮、瘦脸、美白 |
| AI 抠图 | 编辑 →「抠图」→ 自动识别主体 | 换背景、做海报素材 |

### 完整操作流程

```
Step 1: 打开醒图 → 点击「导入」→ 选择相册中刚传输的照片

Step 2: 底部工具栏 →「智能」→「一键精修」
        → 自动分析照片类型（风景/人像/美食）
        → 自动应用 AI 调色方案
        → 左右滑动调整精修强度（0~100）

Step 3: 如需去除杂物
        → 底部工具栏 →「修复」
        → 手指涂抹要消除的区域（路人、垃圾桶等）
        → AI 自动填充背景（1~2 秒完成）

Step 4: 满意后 → 右上角「保存」→ 选择「最高画质」导出
```

### 进阶技巧

- **批量处理**：修好第一张后，长按滤镜 →「复制调整」→ 应用到其他照片
- **模板套用**：首页有大量中文场景模板（日系清新、电影感、氛围感等），一键套用
- **导出设置**：建议导出时选择 **原图尺寸 + 最高画质**，避免压缩损失

## 四、推荐工作流方案

### 方案 A：最佳性价比（免费）

```
相机 → [蓝牙自动传输 JPEG] → 手机 → 醒图/Snapseed AI 精修 → 导出分享
```

**适合**：日常拍照、社交分享、旅行记录

### 方案 B：专业商业级

```
相机 → [Wi-Fi 传输 RAW] → 手机 → Lightroom Mobile AI 降噪+预设批处理 → 导出交付
```

**适合**：商业摄影、产品图、客片交付

### 方案 C：苹果生态

```
相机 → [自动传输] → iPhone → Pixelmator Photo ML 超分+降噪 → 导出
```

**适合**：iPhone 用户、追求简洁操作

## 五、关键注意事项

1. **传输格式选择**：蓝牙自动传输 = JPEG 小图；Wi-Fi 手动传输 = 原图/RAW
2. **AI 消除效果**：简单背景（天空、墙面）效果最好，复杂纹理可能需多次涂抹
3. **商业用途**：建议用 Lightroom Mobile 处理 RAW，精度和色彩管理更专业
4. **醒图 VIP**：免费版 AI 功能已够用，VIP 主要多高级滤镜和去广告
5. **批量自动化**：如需"拍一张自动修一张"，目前无现成方案，需 iOS 快捷指令 + API 自建
