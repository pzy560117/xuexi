# Midscene v1.x 各版本详细 Release Notes

> 按版本倒序排列，记录每个版本的新功能、Bug 修复和其他变更

---

## v1.5.5 (2026-03-16)

### 新功能 🎉
- **Skill CLI**: 支持自定义接口的技能命令行工具
- **GPT-5.4 模型**: 新增 gpt-5.4 模型支持
- **设备健康检查**: 添加显示器检测到设备健康检查
- **Playground**: 提取可复用的 App Shell 和启动器

### Bug 修复 🐞
- 修复 Playground 播放器方向
- 升级 scrcpy-server 到 v3.3.3/v3.3.4
- 支持鸿蒙渲染分辨率屏幕信息解析
- 防止 MCP stdio 服务器成为 100% CPU 的孤儿进程

---

## v1.5.4 (2026-03-11)

### 新功能 🎉
- **iOS**: 通过 bundleId 终止应用
- **MCP 工具导出**: 从所有平台包导出 MidsceneTools
- **CLI 版本**: 添加 CLI 版本支持和健康检查中显示 @midscene/computer 版本
- **动作参数示例**: 在设备动作定义中添加示例参数

### Bug 修复 🐞
- 修复 aiTap 文档声明格式
- iframe 感知的 xpath 和节点缓存
- 保留截图 capturedAt 时间戳

---

## v1.5.3 (2026-03-09)

### 新功能 🎉
- **Remotion Player**: 设备感知的 Shell 渲染
- **AI 调用计时**: 添加 callAi 计时和动作调用前后的计时修复
- **延迟加载图片**: 大型报告延迟加载图片
- **AbortSignal**: aiAct 支持 AbortSignal 取消任务

### Bug 修复 🐞
- 修复 Android 字符输入丢失
- 修复 Linux 上 Chrome 启动
- 修复 Playground ADB 稳定性

---

## v1.5.0 (2026-03-02)

### 新功能 🎉
- **@midscene/harmony**: 鸿蒙 HarmonyOS 自动化包
- **deepThink 恢复**: 恢复子目标规划支持，使用 deepThink 模式
- **详细计时**: 让计时信息更详细

## v1.4.0 ~ v1.4.9 (2026-02-12 ~ 2026-02-28)

### 关键功能
- **Skills 系统**: AI Agent 设备控制技能
- **@midscene/computer-mcp**: 独立桌面自动化 MCP 包
- **Chrome 扩展 MCP**: 后台桥接 MCP 连接
- **deepThink + aiAct**: 深度思考模式融入 act 工具
- **iOS MJPEG 直播流**: Playground 支持 iOS 设备实时流
- **健康检查**: 连接方法中添加健康检查
- **Xvfb 虚拟显示**: Linux 无头桌面自动化
- **Qwen3.5 + doubao-seed 2.0**: 新模型支持
- **自定义截图缩放**: 支持自定义截图收缩比例

## v1.3.0 ~ v1.3.11 (2026-01-27 ~ 2026-02-11)

### 关键功能
- **@midscene/computer**: PC 桌面自动化包
- **LLM 规划 XML 输出**: 规划输出从 JSON 切换到 XML
- **scrcpy 实时截图**: 零缓冲架构 Android 截图
- **sleep 动作**: Action Space 暴露 sleep
- **App 名称映射**: 应用名和包名推理比较
- **cursor move**: 暴露鼠标移动动作
- **AppleScript 输入**: macOS 使用 AppleScript 键盘输入
- **deepLocate**: 新增 aiAct 深度定位选项

### 性能优化
- 异步 uploadTestInfoToServer 消除事件循环阻塞

## v1.0.0 ~ v1.0.4 (2025-12-18 ~ 2025-12-29)

### 里程碑变更
- **纯视觉路径**: Token 降低 80%，不再依赖 DOM
- **多模型组合**: Planning + Insight + Interaction 分别配置
- **MCP 架构重设计**: Action Space 暴露为 MCP 工具
- **API 重命名**: aiAction→aiAct, OPENAI_API_KEY→MIDSCENE_MODEL_API_KEY
- **Gemini-3-Flash**: 新模型支持
- **默认温度**: 设置默认 temperature 为 0
- **Showcases 页面**: 真实场景展示
